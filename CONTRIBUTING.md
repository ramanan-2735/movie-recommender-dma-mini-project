# Contributing to Movie Recommender System

We welcome contributions to improve the movie recommender system! To contribute:

1. **Fork the Repository** to your own account.
2. **Create a Feature Branch** (`git checkout -b feature/recommender-enhancement`).
3. **Commit Changes:** Make sure you don't commit large dataset CSVs or `.pkl` model files (these should be excluded via `.gitignore`).
4. **Open a Pull Request** describing your adjustments.

Thank you!
